<h1>Welcome to pipe-lang</h1>
<p>pipe-lang is a <strong>functional</strong> programming language with <code>Effect&lt;T&gt;</code> for side effects.</p>
<h2>Features</h2>
<p>Here are some key features:</p>
<li>Hindley-Milner type inference</li>
<li>Algebraic data types with pattern matching</li>
<li>First-class functions and closures</li>
<li>Effect system for pure/impure separation</li>
<li>JIT compilation via Cranelift</li>
<hr>
<h2>Code Example</h2>
<pre><code class="language-pipe-lang">
let factorial = (n) => match n {
    0 => 1
    1 => 1
    n => n * factorial(n - 1)
}

let main = () => println(factorial(5))
</code></pre>
<p>The <code>factorial</code> function demonstrates recursion and pattern matching.</p>
